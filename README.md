# 株式会社EIRIA ウェブサイト

株式会社EIRIAのコーポレートサイトです。AIスクール「はじめてのAIスクール 彩成」をはじめとするAI関連事業を紹介しています。

## ファイル構成

```
eiria-website/
├── index.html          # トップページ
├── services.html       # 事業内容
├── company.html        # 会社概要(理念・代表挨拶含む)
├── contact.html        # お問い合わせ
├── privacy.html        # プライバシーポリシー
├── README.md
├── css/
│   └── style.css       # 共通スタイル
├── js/
│   └── main.js         # 共通スクリプト
└── images/
    └── eiria-logo.png  # 会社ロゴ
```

- HTML5 / CSS3 / Vanilla JavaScript
- レスポンシブデザイン対応(PC・タブレット・スマートフォン)
- Google Fonts使用

## 公開方法

GitHub Pagesで無料公開できます。リポジトリ設定の「Pages」から、ブランチを `main`、フォルダを `/ (root)` に指定するだけで公開されます。

## 編集について

- 会社住所、設立年月日、メールアドレスなどは未確定のため、決まり次第 `company.html`、`contact.html` を編集してください
- お問い合わせフォームはダミーです。実際の送信機能を使うには [Formspree](https://formspree.io/) などの外部サービスとの連携が必要です
- 代表者の写真を準備できたら、`company.html` の代表挨拶セクションのビジュアル部分に画像を挿入できます
